# Araba Collective Website

A React site showcasing Araba Collective’s services, work, and brand story.

---

## Tech Stack

- **Framework**: React
- **Build**: Vite
- **Styling**: Tailwind CSS
- **Animations**: GSAP & ScrollTrigger
- **Icons**: lucide-react

---

## Getting Started

### Prerequisites

- Node.js ≥ 16
- npm or Yarn

### Install & Run

```bash
git clone <repo-url>
cd <repo-directory>
npm install
npm run dev
```
